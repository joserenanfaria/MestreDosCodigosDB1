1. Em quais linguagens o C# foi inspirado?

R. O C# tem sua sintaxe baseada na linguagem C++ e possui influência de outras linguagens como Delphi e Java.

2. Inicialmente o C# foi criado para qual finalidade?

R. Possibilitar a criação de soluções na plataforma .NET Framework.

3. Quais os principais motivos para a Microsoft ter migrado para o Core?

R. Tornar o framework multiplataforma e melhorar sua performance.

4. Cite as principais diferenças entre .Net Full Framework e .Net Core.
R. Diferenças:
a. Soluções .NET Framwork são compatíveis apenas com windows e as soluções .NET Core são multiplataforma.
b. .NET Framwork não permite instalações side-by-side e o .NET Core sim.
c. Incompatibilidade entre componentes que utilizam uma versão ou outra.
